SCRIPTS
Los scripts son un programa simple, que generalmente se crean en un archivo de texto.
Para que los scripts funcionen se debe poner en su primera línea:
#!/bin/bash
🖥️
SCRIPT CON ECHO
El comando echo nos sacará por pantalla la información que introduzcamos entre comillas
Ejemplo: Script saludando
bash
#!/bin/bash
#Linea que sera ignorada
echo "Hola"
echo "Soy un script"
echo "salu2"
Hay que darle permisos de ejecución. Para ejecutarlo: ./saludos.sh o source saludos.sh sin Permisos.
📦
SCRIPT CON VARIABLES (I)
Las variables nos guardan el valor en una caja, puede ser de texto o número. Para crear una variable, tenemos que escribir nombrevariable=valor (kilos=80)
Para hacer referencia a una variable tenemos que escribirla con el símbolo $ delante. ($nombre)
bash
#!/bin/bash
#Ejercicio 1 de las variables
nombre=Evelyn
edad=18
estatura=1.60
echo "Hola me llamo $nombre , tengo $edad años y mido $estatura"
SCRIPT CON VARIABLES (II)
Para hacer operaciones con variables, debemos de crear una variable de esta manera:
suma=$[$num1 + $num2]
Ejemplo: suma de datos en una variable.
bash
#!/bin/bash
#ejercicio variables 2
dato1=5
dato2=2
suma=$[$dato1+$dato2]
echo "El resultado de sumar $dato1 y $dato2 es $suma"
SCRIPT CON VARIABLES (III)
Para realizar diferentes operaciones, debemos de escribir lo mismo pero con el signo correspondiente:
resta=$[$num1 - $num2] division=$[$num1 / $num2] multiplicacion=$[$num1 * $num2] cociente=$[$num1 % $num2]
Ejemplo: realizando varias operaciones en variables.
bash
#!/bin/bash
#Ejercicio 3 de variables
dato1=5
dato2=2
suma=$[$dato1 + $dato2]
resta=$[$dato1 - $dato2]
multiplicacion=$[$dato1 * $dato2]
division=$[$dato1 / $dato2]
echo "El resultado de la suma de $dato1 y $dato2 es $suma"
echo "El resultado de la resta de $dato1 y $dato2 es $resta"
echo "El resultado de la multiplicacion de $dato1 y $dato2 es $multiplicacion"
echo "El resultado de la division de $dato1 y $dato2 es $division"

SCRIPT CON READ (INTRODUCIENDO VALORES)
Para que nuestro script pida unos datos al usuario, debemos de crear cajas con el comando read. Solo tenemos que crear la caja escribiendo read nombre. Para hacer referencia a la caja, también escribiremos $nombre
Ejemplo: pide tu nombre, y lo repite.
bash
#!/bin/bash
#ejercicio 4 de variables introduciendo valores

Escript con AND, OR, NOT
Los comandos AND, OR y NOT son expresiones lógicas que se utilizan con IF, WHILE..
EXPRESIONES LOGICAS: AND 🡪 && ó –a 🡪 Verdadero si ambas expresiones son ciertas. OR 🡪 || ó -o 🡪 Verdadero si una de las expresiones es cierta. NOT 🡪 !expresión 🡪 Verdadero si la expresión no es cierta.
Ejercicio 7: Notas y Faltas.
Crea un script que diga que: Estás suspenso si tienes menos de 5 en nota o más de 10 faltas. Estas aprobado si tienes más de 5 y además tienes menos de 10 faltas.
bash
#!/bin/bash
#Ejercicio condicion doble
echo -n "Introduce tu nota: "; read nota
echo -n "Introduce tus faltas: "; read faltas
if [ $nota -ge 5 ] && [ $faltas -lt 10 ]; then
	echo "APROBADO 😁"
fi
if [ $nota -lt 5 ] || [ $faltas -ge 10 ]; then
	echo "SUSPENSO 😔"
fi

SCRIPT CON FOR (BUCLE UN NÚMERO DE VECES)
Para crear un bucle un número de veces específico utilizamos FOR. La estructura de FOR es:
bash
for num in `seq 1 100`;
do
	Aquí escribimos lo que hará durante 100 veces (cada vez num tiene un valor superior al anterior)
done

Los parámetros sirven para introducir los datos cuando ejecutas el script, es decir, sin pedirlos por pantalla.
